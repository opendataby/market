Цены Беларуси

Сравните стоимость товара в магазине и на рынке с его средней ценой по стране и области.

  • Цены по состоянию на март 2016 года указаны в рублях за килограмм, литр, десяток, изделие, данные обновляются ежемесячно
  • Источник данных: Средние цены на товары, реализуемые в розничной сети в магазинах и на городских рынках // Национальный статистический комитет Республики Беларусь
  • Набор данных, используемый в приложении: http://opendata.by/dataset/495/....
  • Комментарий начальника Главного управления статистики цен Национального статистического комитета Республики Беларусь Ярковца А.И. по поводу корректного использования данных о средних ценах
  • Другие проекты сообщества "Открытые данные для Беларуси": opendata.by/projects
  • Контакты сообщества для желающих присоединиться: opendata.by/about
